## Usage

```ts
import { withGuardrails, piiDetector } from 'ai-sdk-guardrails';
import { openai } from '@ai-sdk/openai';

const model = withGuardrails(openai('gpt-4o'), {
  inputGuardrails: [piiDetector()],
});

await generateText({ model, prompt: '...' });
```

## Provider Compatibility

Works with all AI SDK providers: OpenAI, Anthropic, Google, Mistral, Groq, Ollama, and custom providers.

## Quick Example

```ts
import { withGuardrails, piiDetector, sensitiveDataFilter } from 'ai-sdk-guardrails';
import { openai } from '@ai-sdk/openai';
import { generateText } from 'ai';

const model = withGuardrails(openai('gpt-4o'), {
  inputGuardrails: [piiDetector()],
  outputGuardrails: [sensitiveDataFilter()],
});

const { text } = await generateText({
  model,
  prompt: 'Analyze this customer request...',
});
```

## Use Cases

- **Input validation**: Check for PII, prompt injection, toxic content
- **Output validation**: Filter sensitive data, enforce quality standards
- **Compliance**: Block regulated content, maintain audit trails
- **Tool security**: Validate tool calls, enforce execution policies
